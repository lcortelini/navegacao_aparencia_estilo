# Aplicativo teste de estilos
Aplicativo de lista de contatos para teste de estilo, navegação e aparência

- Options menu
- Navigation drawer
- Sobreposição de activitys 
- Toolbar
- Estilos e temas 
- Armazenamento de informações em SharedPreferences
